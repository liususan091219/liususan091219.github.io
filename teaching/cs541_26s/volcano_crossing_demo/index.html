Volcano Crossing

Policy Evaluation Tool

How it works: Define a policy by clicking arrows in each cell to set the agent's intended direction. Adjust the parameters, then sample paths to estimate the policy's value. The agent starts at (2,1), aims for the goal (+40) at (1,4), can collect a small reward (+2) at (3,1), and must avoid the volcanoes (-50) at (1,3) and (2,3). With probability slipProb, the agent slips and moves randomly instead of following the policy.

RL Monte Carlo

Value Estimation Methods

How it works: The agent starts at (2,1) and explores a hidden grid. It does not know where rewards or penalties are — it can only observe (action, reward, next state) after each step. Adjust parameters, then sample episodes to estimate values using Monte Carlo methods. Switch tabs to compare model-based V(s), model-free MC Q(s,a), SARSA (on-policy TD), and Q-learning (off-policy TD).

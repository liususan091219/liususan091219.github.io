Research Interests

Natural language processing, text mining, information retrieval, and their application to software engineering and security are my areas of interest. Here is a list of topics I am working on:

  • Natural language processing for software engineering:
    • Assisting software developers for natural language generation for software testing, information retrieval for NL/code data: LinkSO[NL4SE@FSE 2018], testing NLP models: TestAug [COLING 2022], HateModerate [NAACL 2024].
  • Natural language processing for security:
    • Assisting software developers to explain Android permission purposes: [VL/HCC 2018], CLAP [RE 2018], using NLP to automate security vulnerability ecosystem: [MLHat@KDD 2021], VulLibGen [ACL 2024], etc.
  • Automated machine learning for natural language processing:
    • Assisting data scientists for automated hyperparameter optimization: [ACL-IJCNLP 2021], EcoOptiGen [AutoML 2023], also checkout [FLAML/nlp], our open-source tool for automated hyperparameter optimization for NLP tasks. We are working on research on AutoML for NLP. Check out our KDD 2022 tutorial: Tutorial on FLAML, Our AutoML 2023 paper on Tuning Hyperparameters for large language model.

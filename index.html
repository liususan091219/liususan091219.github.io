Research Interests

My research focuses on NLP for security and software engineering. I develop AI-driven methods to discover, localize, and manage software vulnerabilities at scale. Key areas include:

• Vulnerability Management — AI + program analysis for vulnerability localization, recommender systems for vulnerability discovery, monitoring AI agent security behaviors
• NLP for Software Engineering — mining software artifacts, code understanding, automated testing
• Text Mining & Information Retrieval — content analysis, user behavior modeling

Openings: I am looking for motivated PhD students and post-docs interested in NLP for security/SE. Please email me with your CV.
